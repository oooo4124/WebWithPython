Kakao 광고 등록하기

A. 로그인


회원 가입을 하고 로그인을 하면 첫 화면입니다. 두 번째 메뉴인 "광고관리"를 클릭으로 본격적으로 등록하시면 됩니다.
​

B. 캠페인 등록하기

​

"새 캠페인" 버튼을 클릭해주세요.


1. 캠페인 종류 : 검색 네트워크를 선택하시면 됩니다. 쇼핑플러스, 스타일 포커스는 키워드 광고와 다른 광고 형식입니다.

2. 캠페인명 : 적당한 이름을 적어주세요.

3. 일 예산 :예시로 12,000원으로 잡았습니다

4. 노출 기간 :이벤트성 키워드인 경우 기간을 설정해주시면 되고 아니면 미 설정으로 무한 기간으로 잡으시면 됩니다.

5. 노출 요일/시간 : 해당 키워드가 노출할 시간을 선택해주시면 됩니다.

"저장 후 그룹 등록" 버튼을 클릭하시면 캠페인은 생성이 되고 다음 "그룹 등록"으로 넘어갑니다.


C. 그룹 등록


​ 1. 그룹명 : 적당한 이름을 적어주세요.

2. 광고대상 : "신규 광고대상 등록" 버튼을 눌러 광고할 홈페이지를 등록해주세요.

3. 노출 방식 : 해당 키워드를 검색 시 어느 영역에 노출될지를 선택하시면 됩니다. 저는 보통 다 선택합니다.
정확히 알고 싶으시면 [노출 영역 상세 보기]를 클릭하시면 각 영역에 대한 설명이 있습니다.

4. 소재 노출 방식
랜덤 노출 - 등록된 소재들이 말 그대로 랜덤으로 노출되는 방식입니다.
성과 우선 노출 - 노출된 광고에 어떤 걸 더 많이 클릭하고 사람들이 해당 홈페이지로 이동하는지 데이터를 축척한 뒤,
kakao에서 성과가 있는 걸 우선으로 소재를 노출해주는 방식입니다.

5. 일 예산 : 적당한 예산 범위를 적어 주세요.

6. 확장 검색 : "미 설정"은 내가 정한 키워드에만 광고가 노출되고 확장은 내 키워드와 연관이 높은 경우 추가로 광고가 노출되는 방식입니다. ​



광고대상 클릭은 간단합니다. 웹 사이트명을 입력하고 접속 가능한 URL을 입력해주면 완료됩니다.


D. 키워드 등록, 소재 설정


​ 1.키워드 등록 : 광고 노출을 하고 싶은 키워드들을 입력합니다. 추후 따로 키워드 등록하는 버튼이 있으니 테스트로 2~3개 등록합니다.
그리고 아래 입찰 방식이 있는데요. 키워드 희망 순위를 1~10등 중에 선택하면 자동으로 입찰가가 정해집니다. 직접 입력은 말 그대로 희망하는 입찰가를 선택하시면 됩니다.

2. 소재 설정 : 키워드 등록 시 노출된 홍보 문구를 적어주세요.
- 소재명 : 관리하기 위한 명이니 간단하게 적어주세요.
- 제목 : 노출되는 제목을 적어주시면 됩니다.(15자 이내)
- 설명 문구 : 설명되는 부가 내용을 적어주시면 뒤 빈다.
"확인" 버튼을 클릭하면 키워드 등록까지 마무리됩니다.




승인이 마무리되면 "운영 중"이라고 나오게 되고 키워드 들도 "노출 가능"이라고 나오면서 본격적으로 키워드 광고가 시작하게 됩니다.
